# Peter Luke Digital — Client System

## Structure
```
clients/
  README.md          — this file
  roster.md          — all clients, contacts, fees, status
  dashboard.md       — weekly snapshot: revenue, action items, deadlines
  _template/         — copy this for every new client
  dr-paul/           — symlink → /workspace/group/dr-paul/
```

## Adding a New Client
1. Copy `_template/` → `clients/[client-slug]/`
2. Follow `_template/ONBOARDING.md` checklist
3. Add to `roster.md`
4. Add to `dashboard.md`
5. That's it — same system every time

## Weekly Routine (Mondays)
1. Open `dashboard.md`
2. Update client statuses
3. Check content queue for each client
4. Note any overdue items or upcoming deadlines
5. Save and commit
