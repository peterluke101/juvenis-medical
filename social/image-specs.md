## File Naming Convention
`[client]-[topic]-[platform]-[format]-[WxH].[ext]`

Examples:
- `drpaul-lowt-instagram-square-1080x1080.jpg`
- `drpaul-bhrt-tiktok-cover-1080x1920.jpg`
- `drpaul-peptides-facebook-feed-1200x630.jpg`

## Export Settings
- Format: JPG for photos, PNG for graphics with text
- Quality: 85-90% JPG compression
- Color space: sRGB
- Max file size: 8MB (most platforms limit to 10MB, stay under 8)
